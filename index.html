Once upon a time, I wanted to make a portfolio website...

But the road to this webpage was kinda like riding on a skateboard: I smashed my head a bunch of times hehehehe...


  • HTML

    I first used html to type words and make ugly/simple webpages. Well, technically, I used it to include the different elements of my page in plain text, which means that at the beginning, this webpage use to look very bad.

  • CSS

    Since things were ugly, I decided to learn some CSS to at least try and make this place pretty. Grids and animations took a while to learn but they kinda worked out hehehe...

  • JavaScript

    I then added some JavaScript just because the animations were not the best. I also wanted to make the webpage fancy (add a carousel, loading screen etc). I used JQuery, Bootstrap, Scrollreveal and Owl-Carousel. They animate and do other cool stuff that somehow works.
